## Usage Tips
- Ask for assumptions and data quality checks first.
- Validate outliers before drawing conclusions.
- Request both executive summary and analyst detail views.

> [!IMPORTANT]
> Correlation is not causation. Treat AI-generated insights as hypotheses until validated.

## Quick Prompt Pattern
```text
Dataset Context + Business Question + Metrics + Constraints + Expected Output + Validation Checks
```
