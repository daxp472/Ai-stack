## Usage Tips
- Always ask for sources with publication date.
- Cross-check at least two independent references.
- Separate facts, assumptions, and interpretations.

> [!IMPORTANT]
> AI can hallucinate references. Verify links before using in academic or professional work.

## Quick Prompt Pattern
```text
Question + Scope + Source Type + Date Range + Output Format + Citation Style
```
