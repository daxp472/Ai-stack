## Usage Tips
- Define composition, lens, lighting, and color palette.
- Generate 4-8 variants before selecting finalists.
- Use image-to-image for controlled iteration.

> [!TIP]
> Keep a reusable prompt template with sections for subject, style, camera, lighting, and negatives.

## Quick Prompt Pattern
```text
Subject + Style + Composition + Lighting + Camera + Mood + Negative Prompt
```
