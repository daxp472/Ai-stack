## Where To Find Things

### Category folders (01 to 09)
Each category has:
- `README.md`: curated tools + usage guidance
- `best-prompts.md`: 10 copy-paste prompts
- `tools/README.md`: quick shortlist table
- `workflows/README.md`: implementation workflow

### Docs folder
- `PLATFORM-COMPARISON.md`: practical platform picks and trade-offs
- `MODEL-COMPARISON-2026.md`: model-level view (cost, strengths, limits)
- `FREE-STACK.md`: full free-first stack recommendations
- `AI-usage-policy.md`: responsible use
- `prompt-design-guide.md`: prompt design system

## Suggested Usage Path
1. Start with your use case category.
2. Pick 1 primary tool + 1 backup tool.
3. Use prompts from `best-prompts.md`.
4. Iterate with workflow files.
5. Track your best prompts and add them back via PR.

> [!TIP]
> For fastest results: one tool for idea generation + one tool for execution + one tool for validation.
