# 100% Free Stack Guide (High-Value, No-Cost Start)

This page is for users who want powerful AI workflows with zero subscription spend.

> [!IMPORTANT]
> "Free" can mean quota-limited or feature-limited. Always check latest policy and limits.

## Best Full-Free Starting Stack

| Need | Free Pick | Why it is strong |
|---|---|---|
| General assistant | Gemini free tier | Strong quality with generous dev onboarding |
| Coding inside editor | GitHub Copilot Free | Good baseline completions and chat limits |
| Research with citations | Perplexity Free | Fast source-first answers |
| Automation | n8n self-host/community | Powerful no-license workflow engine |
| Local/private AI | Open-weight models (Mistral/Gemma/Llama) | No per-request vendor cost when self-hosted |

## Free Workflow Recipes

### 1) Student Research Stack (Free)
1. Use Perplexity free for source discovery.
2. Use Gemini free for synthesis notes.
3. Save final prompts in category prompt files.

### 2) Beginner Developer Stack (Free)
1. Code in VS Code with Copilot Free.
2. Use Gemini free/API starter for architecture and docs.
3. Run local open model for repetitive transformations.

### 3) Creator Stack (Free)
1. Script with Gemini free.
2. Generate visuals with free-trial-capable image tools.
3. Edit and publish with free tiers of creator tools.

## Free Tools With Good Value
- Gemini API free tier
- GitHub Copilot Free
- Perplexity Free
- n8n self-host
- Ollama + open-weight models (local)
- Hugging Face Inference free/community offerings

## When To Move From Free To Paid
- You hit rate limits daily.
- You need reliability SLA.
- You need team governance/compliance.
- Output quality gap affects business outcomes.

> [!TIP]
> Use free for experimentation, paid only for stable production paths.
