## Usage Tips
- Start with one high-frequency manual workflow.
- Add observability and error alerts from day one.
- Keep human approval for risky actions.

> [!IMPORTANT]
> Never allow autonomous workflows to execute financial or destructive actions without approval gates.

## Quick Prompt Pattern
```text
Trigger + Input Data + Decision Rules + Actions + Error Handling + Approval Steps
```
