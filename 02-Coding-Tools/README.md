## Usage Tips
- Provide project context, language, and framework before asking for code.
- Request tests and edge-case handling with every generated function.
- Ask for trade-offs before choosing a final implementation.

> [!IMPORTANT]
> Never paste secrets or production credentials into AI tools.

## Quick Prompt Pattern
```text
Stack + Problem + Constraints + Expected Input/Output + Test Cases
```
