# Best Prompts for Video AI

10 ready-to-use prompts for short-form, long-form, and marketing video workflows.

## 1. Viral Reels Script Generator [Beginner]
**Use case:** Write short-form scripts with high retention.

```text
You are a short-form content strategist.
Topic: [TOPIC]
Audience: [AUDIENCE]
Goal: [AWARENESS / LEADS / SALES]
Duration: 30-45 seconds

Create 3 reels scripts with:
- 1-line hook
- 3 value beats
- 1 proof element
- 1 CTA

Tone: punchy, human, not clickbait.
```

## 2. Shot List from Script [Intermediate]
**Use case:** Turn a script into a production-ready shot plan.

```text
Act as a video director.
Script:
[PASTE SCRIPT]

Generate a detailed shot list:
- Shot number
- Camera angle
- Movement
- Scene duration
- On-screen text
- B-roll suggestions
- Transition type

Optimize for mobile-first viewing.
```

## 3. Long Video to 10 Shorts [Advanced]
**Use case:** Repurpose one long video into multiple clips.

```text
You are a content repurposing editor.
Transcript:
[PASTE TRANSCRIPT]

Identify 10 short clip opportunities.
For each clip provide:
- Timestamp range
- Hook line
- Clip title
- Subtitle style suggestion
- CTA line

Prioritize moments with strong opinion, story, or tactical insight.
```

## 4. YouTube Retention Optimizer [Advanced]
**Use case:** Improve retention curve before publishing.

```text
You are a YouTube strategist.
Video script:
[PASTE SCRIPT]
Target audience: [AUDIENCE]

Audit script for:
1) Weak opening segments
2) Drop-off risk points
3) Missing pattern interrupts
4) CTA placement quality

Then rewrite with retention improvements and explain each change.
```

## 5. Ad Video Concept Builder [Intermediate]
**Use case:** Create high-converting ad concepts quickly.

```text
Act as a performance marketing creative director.
Product: [PRODUCT]
Audience pain points: [PAINS]
Offer: [OFFER]
Platform: [META/TIKTOK/YOUTUBE]

Generate 5 ad concepts with:
- Hook
- Visual concept
- Script angle
- Offer reveal point
- CTA

Make ideas specific and testable.
```

## 6. B-Roll Idea Engine [Beginner]
**Use case:** Add visual richness to talking-head videos.

```text
You are a B-roll planner.
Video topic: [TOPIC]
Talking points:
[PASTE BULLETS]

Return 20 B-roll ideas mapped to each talking point.
Include shot style, location, and mood.
Prefer low-budget, creator-friendly options.
```

## 7. Interview Edit Assistant [Intermediate]
**Use case:** Speed up interview editing decisions.

```text
Act as an interview editor.
Transcript:
[PASTE TRANSCRIPT]
Objective: [EDUCATE / ENTERTAIN / PROMOTE]

Provide:
- Best quotes to keep
- Segments to trim
- Logical chapter breaks
- Suggested lower-thirds text
- Title and thumbnail angles
```

## 8. Video SEO Metadata Pack [Beginner]
**Use case:** Optimize discoverability on YouTube.

```text
You are a YouTube SEO assistant.
Video topic: [TOPIC]
Target keyword: [KEYWORD]
Transcript summary: [SUMMARY]

Generate:
- 10 title options
- SEO description
- Tags list
- 3 pinned comment ideas
- 5 chapters with timestamps format

Keep language natural and non-spammy.
```

## 9. Avatar Presenter Script [Intermediate]
**Use case:** Create polished scripts for AI avatar videos.

```text
Act as a corporate communication writer.
Audience: [EMPLOYEES/CUSTOMERS/PARTNERS]
Message goal: [GOAL]
Duration: [60-120 sec]

Write an avatar-friendly script with:
- Clear opening
- Short, speakable sentences
- Professional tone
- Key message repetition
- CTA with next steps
```

## 10. Multilingual Dubbing Prompt [Advanced]
**Use case:** Localize videos while preserving tone.

```text
You are a localization editor.
Original script:
[PASTE SCRIPT]
Target languages: [LIST]
Brand tone: [TONE]

Create localized versions that preserve:
- Intent
- Emotional tone
- Cultural relevance
- CTA clarity

Flag phrases that require adaptation rather than direct translation.
```
