## Usage Tips
- Start from a shot list, not a generic idea.
- Ask AI for hook options before finalizing script.
- Design edits for retention: pace, subtitle rhythm, and scene changes.

> [!TIP]
> Build a reusable "video brief" template with audience, offer, duration, and tone.

## Quick Prompt Pattern
```text
Audience + Goal + Story Arc + Visual Style + Shot Sequence + CTA
```
