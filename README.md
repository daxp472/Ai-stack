## Best Platform by Use Case

| Use Case | Best Primary Pick | Why it wins | Strong Alternative |
|---|---|---|---|
| Coding assistant inside IDE | GitHub Copilot | Tight IDE integration + high velocity completions | Cursor |
| API-based product build | OpenAI API | Broad multimodal stack + tools ecosystem | Gemini API |
| Long-context writing and reasoning | Claude (Sonnet/Opus line) | Excellent quality for long documents and nuanced writing | ChatGPT |
| Research with citations | Perplexity | Fast citation-first workflow with multi-model orchestration | Gemini + Search grounding |
| Lowest cost high-volume API | Gemini Flash/Flash-Lite family | Strong price-performance in volume workloads | Grok 4.1 Fast |
| Fast visual ideation | Midjourney / Leonardo | Strong creative image output styles | GPT Image |
| Workflow automation | Make / n8n | Reliable automations and connectors | Zapier |

> [!TIP]
> Your best stack is usually two layers: one daily chat platform + one API platform for production workloads.

## Model Comparison Snapshot

| Provider | Model Family (examples) | Typical Strength | Cost Level | Free Access Signal |
|---|---|---|---|---|
| OpenAI | GPT-5.4, GPT-5.4 mini/nano, GPT-5.3-Codex | Strong coding, agentic tools, multimodal depth | Medium to High | API is pay-as-you-go; Chat products have separate plans |
| Anthropic | Claude Opus 4.6, Sonnet 4.6, Haiku 4.5 | High quality reasoning and long-form quality | Medium to High | Free chat tier exists, API is paid |
| Google | Gemini 2.5 Pro/Flash/Flash-Lite, Gemini 3.x previews | Best price-performance at scale + grounding options | Low to Medium | Developer free tier with generous starting access |
| xAI | Grok 4.20, Grok 4.1 Fast | Real-time search angle + fast variants | Low to Medium | Free chat access varies; API is paid |
| Mistral | Open + premier model mix (Small/Large/Devstral/Codestral) | Open-weight flexibility and self-host options | Low to Medium | Open models available; paid hosted tiers for premier |

> [!IMPORTANT]
> Pricing and limits change frequently. Use this repo as decision guidance and verify current values from official links in [docs/PLATFORM-COMPARISON.md](docs/PLATFORM-COMPARISON.md).

## Free-First Track

If budget is zero, start here:
- Gemini API free tier for experimentation and prototypes
- GitHub Copilot Free for starter coding assistance
- Open-weight Mistral/Gemma/Llama routes via local/self-host paths
- Perplexity free tier for citation-first quick research
- n8n community/self-host for automation workflows

Full guide: [docs/FREE-STACK.md](docs/FREE-STACK.md)
