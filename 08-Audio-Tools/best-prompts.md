# Best Prompts for Audio AI

10 ready-to-use prompts for voice, transcription, and audio production.

## 1. Podcast Episode Script Builder [Beginner]
**Use case:** Draft a complete podcast episode quickly.

```text
You are a podcast scriptwriter.
Episode topic: [TOPIC]
Audience: [AUDIENCE]
Episode duration: [MINUTES]

Create:
- Hook intro
- 3 core segments
- Transition lines
- Audience engagement question
- Outro CTA

Tone: conversational, clear, and energetic.
```

## 2. Voiceover Performance Prompt [Intermediate]
**Use case:** Generate natural-sounding TTS scripts.

```text
Act as a voiceover director.
Script content:
[PASTE SCRIPT]
Voice style target: [WARM/CONFIDENT/CALM]
Pacing: [SLOW/MEDIUM/FAST]

Rewrite script for speech clarity:
- Short breath-friendly lines
- Emphasis markers
- Pronunciation notes
- Pause cues
```

## 3. Audio Cleanup Workflow [Beginner]
**Use case:** Improve low-quality recordings.

```text
You are an audio post-production assistant.
Recording issues: [NOISE, ECHO, CLIPPING, LEVELS]
Recording context: [ROOM, MIC, DISTANCE]

Provide a step-by-step cleanup chain:
- Noise reduction
- EQ adjustments
- Compression settings
- De-esser suggestions
- Loudness target recommendations
```

## 4. Interview Transcript Editor [Intermediate]
**Use case:** Turn raw transcripts into readable content.

```text
Act as a transcript editor.
Transcript:
[PASTE]

Produce:
- Cleaned transcript with punctuation
- Remove filler words while preserving meaning
- Timestamp cue points for key moments
- Pull 10 quotable highlights
```

## 5. Audiobook Narration Prep [Advanced]
**Use case:** Prepare long-form narration scripts for consistency.

```text
You are an audiobook narration coach.
Text:
[PASTE CHAPTER]

Create:
- Character voice style notes
- Pronunciation guide for names/terms
- Emotional pacing map by paragraph
- Pause and emphasis suggestions
- Consistency checklist for chapter-to-chapter delivery
```

## 6. Multilingual Voice Localization [Advanced]
**Use case:** Localize spoken content across languages.

```text
Act as a localization voice director.
Original script:
[PASTE SCRIPT]
Target languages: [LIST]
Brand tone: [TONE]

Deliver for each language:
- Localized script
- Cultural adaptation notes
- Pronunciation cautions
- Alternative phrasing for difficult lines
```

## 7. Music Bed Recommendation Prompt [Beginner]
**Use case:** Match background music to content mood.

```text
You are a sound design assistant.
Content type: [PODCAST/YOUTUBE/AD]
Desired mood arc: [MOOD TIMELINE]
Audience: [AUDIENCE]

Suggest:
- 5 music style directions
- BPM range
- Instrumentation profile
- Where to rise/fall intensity
- Ducking strategy with voice
```

## 8. YouTube Voiceover Script Optimizer [Intermediate]
**Use case:** Improve script flow for narration videos.

```text
Act as a narrative script editor.
Draft voiceover:
[PASTE]

Optimize for:
- Spoken clarity
- Story progression
- Emotional rhythm
- Smooth transitions
- Strong ending CTA

Keep total word count within [TARGET WORDS].
```

## 9. Pronunciation Dictionary Builder [Advanced]
**Use case:** Standardize pronunciation in recurring series.

```text
You are a speech consistency specialist.
Terms and names list:
[PASTE]

Build a pronunciation dictionary with:
- Phonetic spelling
- Stress markers
- Common mistakes to avoid
- Context examples
- Quick rehearsal script
```

## 10. Audio QA Checklist Generator [Intermediate]
**Use case:** Final quality check before publishing.

```text
Act as an audio QA engineer.
Project type: [PODCAST/COURSE/AD]
Technical target: [LOUDNESS, FORMAT]

Create a release checklist for:
- Noise floor
- Peak levels
- Loudness compliance
- Stereo balance
- Intro/outro consistency
- Metadata completeness
```
