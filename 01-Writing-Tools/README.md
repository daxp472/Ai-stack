## Usage Tips
- Always provide audience, format, and tone in the first line.
- Ask for 3 variations, then merge the strongest pieces.
- Use a second pass prompt for fact-checking and simplification.

> [!TIP]
> Best results come from feeding examples of your own writing and asking the model to match your style constraints.

## Quick Prompt Pattern
```text
Role + Audience + Goal + Constraints + Voice + Output Format
```
