# Best Prompts for Productivity AI

10 ready-to-use prompts to plan better and execute with less stress.

## 1. Weekly Planning Sprint [Beginner]
**Use case:** Plan your week with realistic priorities.

```text
You are my execution coach.
My goals this week:
[PASTE GOALS]
Available hours: [HOURS]
Constraints: [MEETINGS/DEADLINES]

Create:
- Top 3 priorities
- Daily task plan
- Deep work blocks
- Buffer time recommendations
- What to defer
```

## 2. Daily Focus Planner [Beginner]
**Use case:** Start each day with clarity.

```text
Act as a productivity assistant.
Today's commitments:
[PASTE TASKS]
Energy level: [LOW/MED/HIGH]

Build:
- 3 must-do tasks
- Time-blocked schedule
- Break plan
- Distraction prevention rules
- End-of-day review checklist
```

## 3. Meeting-to-Action Converter [Intermediate]
**Use case:** Turn messy meeting notes into execution tasks.

```text
You are an operations assistant.
Meeting notes:
[PASTE NOTES]

Extract:
- Decisions made
- Action items (owner + due date)
- Risks and blockers
- Follow-up questions
- Next meeting agenda draft
```

## 4. SOP Draft Builder [Intermediate]
**Use case:** Document repeatable processes fast.

```text
Act as a process designer.
Process name: [PROCESS]
Inputs: [INPUTS]
Tools used: [TOOLS]
Desired output: [OUTPUT]

Write an SOP with:
- Objective
- Step-by-step instructions
- Quality checklist
- Common failure points
- Escalation path
```

## 5. Procrastination Breaker [Beginner]
**Use case:** Start difficult tasks with momentum.

```text
You are a behavioral productivity coach.
Task I am avoiding: [TASK]
Why I am stuck: [REASON]
Deadline: [DATE]

Help me by creating:
- 10-minute starter task
- 3 micro-milestones
- Accountability trigger
- Reward loop
- Fallback plan if I slip
```

## 6. Second Brain Organizer [Advanced]
**Use case:** Organize notes into a searchable system.

```text
You are a knowledge management architect.
Current note dump:
[PASTE NOTES]

Design:
- Folder/tag taxonomy
- Naming conventions
- Capture template
- Weekly review system
- Retrieval workflow by project and topic
```

## 7. Deadline Recovery Plan [Advanced]
**Use case:** Recover when a project is behind schedule.

```text
Act as a project rescue manager.
Project status:
[PASTE STATUS]
Missed milestones: [LIST]
Hard deadline: [DATE]

Create:
- Recovery roadmap
- Scope cuts vs must-keep matrix
- Communication update draft
- Risk mitigation plan
- Daily checkpoint format
```

## 8. Decision Journal Prompt [Intermediate]
**Use case:** Improve decision quality over time.

```text
You are my decision advisor.
Decision context: [CONTEXT]
Options: [OPTION A/B/C]

Help me evaluate:
- Assumptions
- Best/worst case
- Reversible vs irreversible aspects
- Decision criteria scoring
- Final recommendation with rationale
```

## 9. Personal KPI Dashboard Builder [Advanced]
**Use case:** Track productivity with meaningful metrics.

```text
Act as a personal analytics coach.
Focus areas: [STUDY/WORK/HEALTH/CONTENT]
Current habits and output:
[PASTE]

Define:
- 5 leading indicators
- 5 lagging indicators
- Weekly dashboard template
- Reflection questions
- Adjustment rules when metrics drop
```

## 10. Burnout Prevention Check-in [Intermediate]
**Use case:** Maintain sustainable performance.

```text
You are a performance and wellbeing coach.
Current workload: [DETAILS]
Stress signals: [DETAILS]
Sleep and recovery pattern: [DETAILS]

Create:
- Workload risk assessment
- Immediate relief actions
- Weekly energy management plan
- Boundary-setting scripts
- Escalation signs I should not ignore
```
