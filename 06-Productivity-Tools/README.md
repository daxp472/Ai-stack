## Usage Tips
- Turn goals into weekly priorities and daily actions.
- Use AI for planning first, then execution tracking.
- Keep one source of truth for tasks.

> [!TIP]
> A short daily AI review prompt can save hours of context switching.

## Quick Prompt Pattern
```text
Goal + Timeframe + Constraints + Current State + Next 3 Actions + Check-in cadence
```
